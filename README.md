# WisdomFishTank
一款软硬件结合的智慧鱼缸项目



### 硬件与软件开发环境：

#### 硬件：树莓派、USB摄像头、若干传感器、ESP8266

#### 软件：IDEA、Android Studio、Arduino



###  功能：

- 通过传感器获取鱼缸的温度、光照、含氧量、浑浊度、PH值等信息
- 通过传感器获取的信息进行换水、打开氧泵等逻辑开关
- 通过摄像头获取鱼塘信息并进行推流，在Android端或者web端进行监控查看
- Android端与web端可视化鱼缸信息
